# Login-Form

### Opis
Ovaj projekat je kreiran za prijavu/registraciju korisnika. Svrha mu je da demonstrira moju sposobnost kreiranja i shvatanja koncepta uređivanja i dodavanja korisnika

### Korištene Tehnologije (Tech Stack)
* **Back-end:** HTML 5, CSS 3, JavaScript ES5
* **Alati/** Git

### Funkcionalnosti
* Prijava Korisnika
* Registracija Korisnika
  

### Pokretanje Projekta (Lokalna Instalacija)

Slijedite ove korake da biste pokrenuli aplikaciju:
1.  Klonirati repozitorij: `git clone https://github.com/mirzadev2000/projects`
2.  Otvoriti projekat u svom IDE-u (VS Code).
3.  Pokrenuti klasu Index.html za start Web Aplikacije
