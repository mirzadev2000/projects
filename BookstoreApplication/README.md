# BookStore App

### Opis
Ovaj projekat je kreiran za dodavanje/uređivanje/brisanje knjiga iz aplikacije. Svrha mu je da demonstrira moju sposobnost kreiranja i razumijevanja koncepta Java Programiranja.

### Korištene Tehnologije (Tech Stack)
* **Back-end:** Java 17

### Funkcionalnosti
* Kreira knjige u sistem
* Dodavanje/Uređivanje/Brisanje knjiga iz sistema 

### Pokretanje Projekta (Lokalna Instalacija)

Slijedite ove korake da biste pokrenuli aplikaciju:
1.  Klonirati repozitorij: `git clone https://github.com/mirzadev2000/projects`
2.  Otvoriti projekat u svom IDE-u (IntelliJ ili Eclipse).
3.  Pokrenuti klasu [Main] za start Java aplikacije.
