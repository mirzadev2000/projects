# Wikipedia Clone Site (JavaScript API za Biblioteku)

### Opis
Ovaj projekat je kreiran za pretraživanje Wikipedia članaka na Engleskom jeziku. Svrha ovog projekta je pretraživanje članaka wikipedije na engleskom jeziku.Korišten je API za povezivanje sa Wikipedia website-om na engleskom jeziku i domenom na engleskom 

### Korištene Tehnologije (Tech Stack)
* **Front-end:** HTML 5,CSS 3,JavaScript
* **Alati:** Git (za testiranje API-ja)

### Funkcionalnosti
* Kucati u Wikipedia tražilici
* Kliknuti na bilo koji članak na wikipediji
* Kada kliknete automatski vas prebacuje na web-site od članka koji ste tražili u aplikaciji

### Pokretanje Projekta (Lokalna Instalacija)

Slijedite ove korake da biste pokrenuli aplikaciju:
1.  Klonirajte repozitorij: `git clone https://github.com/mirzadev2000/projects`
2.  Otvoriti projekat u IDE-u (VS Code).
3.  Pokrenuti klasu HTML Index.html u VS Code-u za pokretanje aplikacije (Preko Liveservera).
