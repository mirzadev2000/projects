# Vrijeme/Weather App (npr. JavaScript API za Biblioteku)

### Opis
Ovaj projekat je kreiran za povezivanje sa API-em te provjeravanje vremenske prognoze u svim gradovima. Svrha mu je da demonstrira moju spoospbnost,kreiranje te razumijevanje API-a i Front-end programiranja

### Korištene Tehnologije (Tech Stack)
* **Front-end:** HTML 5, CSS 3, JavaScript ES5
* **Alati/Ostalo:** Git,  (za testiranje API-ja)

### Funkcionalnosti
* Dohvata sve gradove svijeta
* Informacije kolika je temperatura
