# Calendar App 

### Opis
Ovaj projekat je kreiran za praćenje datuma i kreiranje podsjetnika za događaje. Svrha mu je da demonstrira moju sposobnost kreiranja i razumijevanja koncepta programiranja u Front-endu.

### Korištene Tehnologije (Tech Stack)
* **Front-end:** HTML 5, CSS 3, JavaScript ES5
* **Alati/Ostalo:** Git (za testiranje API-ja)

### Funkcionalnosti

* Klikanje na datum bilo koji ili klikanje
* Dodavanje događaja i opis
* Sačuvanje događaja u aplikaciji

### Pokretanje Projekta (Lokalna Instalacija)

Slijedite ove korake da biste pokrenuli aplikaciju:
1.  Klonirati repozitorij: `git clone https://github.com/mirzadev2000/projects`
2.  Otvoriti projekat u svom IDE-u (VS Code).
3.  Pokrenuti klasu HTML za start Web Aplikacije.
